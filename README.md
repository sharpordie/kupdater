# <samp>GUIDANCE</samp>

## One-command execution

Ensure your device is plugged and launch this command.

```shell
/bin/zsh -c "$(curl -fsSL https://raw.githubusercontent.com/sharpordie/kupdator/HEAD/src/kupdator.sh)"
```
